# Прогнозирование температуры стали

Выпускной проект курса "Специалист по DataSciene". 

**Постановка задачи:**  
- Построение модели, которая предскажет температуру стали  

**Исходные данные:**  
- Набор файлов из различных источников  

**Требования по качеству моделей:**  
- Метрика MAE<=6 на тестовой выборке

**Область применения:**  
- Промышленность

**Навыки и инструменты:**
pandas, numpy, seaborn, matplotlib, sklearn, xgboost, catboost
