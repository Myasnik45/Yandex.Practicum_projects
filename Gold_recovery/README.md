# Прогнозирование восстановления золота из руды

Сборный проект темы "Обучение с учителем"

## Постановка задачи
Необходимо подготовить прототип модели машинного обучения для компании «Цифра». Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды.

## Исходные данные
    Параметрами добычи и очистки;
    Фиксированная обучающая и тестовая выборки;
    Описание технологического процесса.


**Навыки и инструменты:**
pandas, seaborn, scipy, matplotlib, sklearn
